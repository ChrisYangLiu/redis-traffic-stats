# NAME

App::redis\_traffic\_stats - Redis query analyzer for counting, traffic stats by command

# INSTALLATION

To install this module, run the following commands:

    perl Build.PL
    ./Build
    ./Build test
    ./Build install

# DESCRIPTION

See [redis-traffic-stats](https://metacpan.org/pod/redis-traffic-stats).
